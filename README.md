# customer-churn-prediction

## GOAL
- Predict if a customer will churn(stop using the store/service)
- Find what features affect the customer retention and which do not affect it.
- clean data and make sure all data is sutaible for use in models
- create a presentation to present data in power BI

## OUTLINE
- Data collection 
    - Telco Customer Churn -Kaggle
- Exploratory Data Analysis
    - check for missing data, features present, unique values
    - Univariate analysis -> distributions of key variables 
    - Bivariate Analysis -> comparism of churned customers, heatmaps
- Data Preprocessing
    - handle missing values
    - convert categorical var - one hot encoding
    - normalise features
    - split into training, validation and testing sets
    - address imbalance
    - hyperparameter tuning
- model building
    - baseline models
    - advance models
    - evaluation
- Discussion
- Visualization & dashboarding
